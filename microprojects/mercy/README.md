# Alfred :: Support triage

A triage assistant for support teams and founders handling inbound questions.
Alfred answers, diagnoses, and drafts actionable tickets when needed.

## Features

- Searches docs (Mintlify) for grounding
- Checks GitHub issues for regressions
- Drafts Linear tickets with reproduction steps and suspected cause
